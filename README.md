# Mi Proyecto

Este es un ejemplo de cómo ejecutar código HTML en un archivo README.md.

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo HTML</title>
</head>
<body>
    <h1>¡Hola, Mundo!</h1>
    <p>Este es un párrafo de ejemplo en HTML.</p>
</body>
</html>
